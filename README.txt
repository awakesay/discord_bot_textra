概要
    日本語<->英語双方向に翻訳するDiscord botです。

事前準備
    Discord botのAPIキー（/config/discord_bot.json)
    TexTraサービスのAPIキー（/config/textra_param.json）
    日本語話者・英語話者のDiscord ID（/config/speakers.json）

動作環境
    PC: Raspberry Pi 4
    OS: Raspberry pi OS

言語・パッケージ
    Python 3.9.2
    pycord
    requests

サービス
    みんなの自動翻訳@TexTra
    URL: https://mt-auto-minhon-mlt.ucri.jgn-x.jp/
